# Script Bảo trì

Tài liệu này mô tả các script dùng để vận hành hoặc bảo trì **[Tên Dự án]**.

## Danh mục Script

| Script | Mục đích | Khi nào chạy |
| --- | --- | --- |
| `[đường dẫn script]` | `[Mô tả]` | `[Thời điểm]` |

## Lệnh thường dùng

```bash

```
